# PRD Securability Enhancement Example

This folder provides a concrete before/after example for the PRD enhancement workflow.

## Files

- `input-prd.md` — Baseline PRD excerpt before ASVS and FIASSE/SSEM enhancement.
- `enhanced-prd.md` — Enhanced PRD output with:
  - ASVS level decision
  - Feature-to-ASVS mapping
  - Requirement gap closure
  - SSEM attribute annotations
  - FIASSE tenet annotations

## How to Use

1. Review `input-prd.md`.
2. Compare against `enhanced-prd.md`.
3. Use the same transformation pattern with `/prd-securability-enhance` on your own PRD.
